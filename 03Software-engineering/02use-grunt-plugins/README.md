## 插件使用

一般插件的命名规范为`grunt-contrib-xxx`

运行的时候，对于的名字就是 xxx  即使用

```zsh
yarn grunt xxx
```



常用plugins

- [grunt-sass](https://www.npmjs.com/package/grunt-sass)
- [grunt-babel](https://www.npmjs.com/package/grunt-babel)
- grunt-contrib-watch
- load-grunt-tasks



本项目实现了 scss -> css js代码低版本兼容 监听修改，自动更新网页，方便开发
